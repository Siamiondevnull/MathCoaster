# Project Brief: MathCoaster

## Overview

MathCoaster - development project with ASP.NET Core backend and frontend.

## Structure

- **back-end/MathCoasterApi** - ASP.NET Core Web API (.NET 10)
- **front-end** - Frontend application
- **cursor-memory-bank** - Memory Bank workflow system
- **.cursor** - Cursor rules and commands

## Goals

- MathCoaster: physics-based puzzle where the player defines y=f(x) to build a rail; a ball rolls under gravity (MVP delivered).
- Frontend: React SPA with Matter.js and HTML5 Canvas; API client prepared for future backend.

## Constraints

- Use English only in code and literals (project rule)
- Platform: Windows
